# [1979](https://leetcode.com/problems/find-greatest-common-divisor-of-array/)
```python
def main(nums):
   mini, maxi = min(nums), max(nums)
   while mini:
      maxi, mini = mini, maxi % mini
   return maxi
```
#### Assumption: N = the number of elements
#### Complexity: runtime = O(N), space = O(1)
